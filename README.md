### npm i 包

- --save
  - 使用不使用--save都会将配置信息记录到dependencies中，项目运行时所需依赖。
- --save-dev  -D
  - 使用--save-dev会将配置信息记录到devDependencies中，这是在开发过程中需要用到的依赖，发布时用不到。
